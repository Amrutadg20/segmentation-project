# Customer Segmentation Project

## Overview
Segments customers based on behavior and demographics using K-Means clustering, 
built as part of the Thiranex Data Analytics Internship (Task 1).

## Objective
Segment customers based on income, spending patterns, purchase frequency, and 
recency to generate targeted marketing insights.

## Tools Used
- Python (pandas, scikit-learn, matplotlib, seaborn)
- Jupyter Notebook
- K-Means Clustering

## Key Steps
1. Data cleaning and EDA
2. Feature scaling
3. Elbow method to determine optimal clusters (k=4)
4. K-Means clustering
5. Segment profiling and business insights

## Results
Identified 4 customer segments:
- Premium Loyal Customers
- Lapsed High-Value Buyers
- Active Budget Shoppers
- Dormant/Churned Customers

## Files
- `Customer_Segmentation_Project.ipynb` — full analysis notebook
- `customer_segmentation_sample.csv` — input dataset
- `segmented_customers_output.csv` — output with cluster labels
